# Eng+
Private startup project for language learners

## Features

* SSR using Next.js
* Reusable components
* Adjustable design with Bootstrap

## Built With

* [React v16.13.1](https://reactjs.org/) - JS library
* [Next v9.5.2](https://nextjs.org/) - React framework for production
* [Bootstrap v4.5](https://getbootstrap.com/) - UI library
* [Sass](https://sass-lang.com/) - CSS extension language
